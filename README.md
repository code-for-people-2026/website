# Code For People Website

为“工友敲键盘 / 数据平权、AI 下乡”准备的官方网站。

## Tech Stack

- Next.js App Router
- shadcn-style local UI components
- Payload CMS
- Vercel deployment target

## Development

```bash
pnpm install
pnpm dev
```

Public site: http://localhost:3000

Payload admin: http://localhost:3000/admin
